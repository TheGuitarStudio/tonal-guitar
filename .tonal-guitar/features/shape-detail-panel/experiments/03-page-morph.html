Shape Library

Page-morph: the index is a grid of chord entities ("Em7", "Cmaj7") showing which voicing families exist for each. Clicking one morphs the whole page into that chord's view — full musical context up top, every registered voicing below grouped by family, with dashed tags marking families that have no registered shape yet.
